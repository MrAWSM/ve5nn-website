# VE5NN — Regina Amateur Radio Association

Website for the Regina Amateur Radio Association Inc. (VE5NN).

**Live site:** https://mrawsm.github.io/ve5nn-website/

## About

A static website for the VE5NN amateur radio club in Regina, Saskatchewan, Canada. Built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no build tools, no dependencies beyond Google Fonts.

## Pages

- **Home** — Welcome, meeting info, quick repeater summary, contact
- **About** — Club history, membership, meetings, address
- **Nets** — HF and VHF net schedules
- **Repeaters** — Area repeater table (frequency, offset, tone)
- **CW** — Morse code club (coming soon)
- **Links** — Curated ham radio resources by category
- **Gallery** — Club event photos

## Tech Stack

- HTML5 (semantic, accessible)
- CSS (custom properties, no frameworks)
- Vanilla JavaScript (~120 lines total)
- Deployed via GitHub Pages from `main` branch
